# 零收入阶段免费技术栈（UE5.8 原生优先）

> **性质**: 项目**选定**的技术栈与获取方式，不是「已在广州地图上验收通过」的证明。  
> **引擎钉**: Unreal Engine **5.8**（见 [ENGINE_STRATEGY.md](ENGINE_STRATEGY.md)）；UE6 发布后整仓替换。  
> **费用原则**: 零收入阶段优先 **引擎原生内置** + **明确免费商用** 的辅助工具；授权以 **Epic / 各工具官方条款** 为准（勿把二手摘要当合同）。  
> **本仓库状态图例**: 引擎原生 | 免费条款依赖 | **配置声明** / **代码草稿** / **缺失** / **可选扩展**

相关：[REALITY_STATUS.md](REALITY_STATUS.md) · [ROADMAP_3A.md](ROADMAP_3A.md) · [README.md](../README.md)

## 核心前提

| 项 | 说明 |
|----|------|
| 营收阶段 | 当前按**零收入 / 学习研究**路径选型；商业发布前必须复核 Epic EULA 与资产授权 |
| 能力来源 | 渲染、大世界、物理、人群等优先调用 **UE5.8 自带模块**，少绑第三方付费中间件 |

| 硬性性能目标（未达成） | 常驻内存等指标（如 ≤4.2GB）仅为**目标**，未在真机真图验证 |

## 一、渲染管线（UE5.8 原生）

| 技术 | 用途（项目意图） | 类型 | 本仓库状态 |
|------|------------------|------|------------|
| **Nanite** | 高模建筑/雕塑流送与自动 LOD | 引擎原生 | **配置声明**（CVar / FeatureFlags）；无 Content 网格 |

## 三、物理模拟（Chaos 优先）

零收入阶段：**Chaos 为默认物理路径**（引擎自带）。

| 技术 | 用途（项目意图） | 类型 | 本仓库状态 |
|------|------------------|------|------------|
| **Chaos** 破坏 / 约束 | 城市互动破坏等 | 引擎原生 | **目标**；历史配置曾倾向 Jolt 替换 |
| **Chaos Vehicles** | 载具动力学 | 引擎原生 | **目标**；现有车辆多为 **代码草稿** |
| Niagara 流体/粒子耦合 | 雨、烟尘、水体表现辅助 | 引擎原生 | 插件启用声明；**缺失**资产与联调 |
| **JoltPhysics 插件** | 第三方物理（可选） | 可选包装 | **默认 Disabled**；无完整二进制 |

说明：仓库内仍保留 Jolt 相关 C++/插件源码以便日后对比，但 **零收入默认不启用**，避免非原生依赖挡住有引擎环境的打开流程。

## 启用顺序（有 UE5.8 的环境时）

1. 仅引擎原生插件打开工程（Jolt/SoLoud 保持 Disabled）。  
2. 空关卡验证 Nanite/Lumen/TSR 等（M1b / M2）。  
3. 再导入 Megascans / MetaHuman / Blender 资产做 M3 CBD 切片。  
4. 第三方物理/音频仅在有明确收益且可免费构建时再 `Enabled: true`。
